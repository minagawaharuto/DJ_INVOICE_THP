## やったこと

### 1. 請求書自動転記システムの構築
GmailのPDF請求書を自動でスプレッドシートに転記するGASシステムを新規作成した。

**作成ファイル:**
- `gas/Code.gs` - メイン処理・トリガー設定・テスト関数
- `gas/GeminiApi.gs` - Gemini APIでPDFを解析
- `gas/SheetManager.gs` - スプレッドシートへの転記処理
- `gas/MailNotifier.gs` - エラー・超過通知メール
- `gas/appsscript.json` - GASマニフェスト
- `.clasp.json` - claspとGASプロジェクトの連携設定

### 2. 動作確認・バグ修正

| 問題 | 対応 |
|------|------|
| `setupTrigger` で権限エラー | `appsscript.json` に `script.scriptapp` スコープを追加 |
| Geminiモデルが見つからない | `gemini-1.5-flash` → `gemini-2.5-flash` に変更 |
| 転記先が200行以降になる | `getLastRow()` をやめてB列を上から走査する方式に変更 |
| 1列ずれて転記される | 既存シートがB列始まりのため、書き込み開始列を2に修正 |
| `runOnce` でメールが見つからない | `debugGmailSearch` 関数を追加して検索状態を確認できるようにした |

### 3. 仕様変更・機能追加

**芸名の抽出ルール変更**
- 括弧内ではなく、改行された次の行を芸名として認識するよう変更

**請求書の集計ロジック変更**
- 明細を月ごとに集計して1行に転記（複数月は複数行）
- 金額 = 源泉徴収後の合計金額（合計金額(内税)）
- 請求月日 = 「4月」のような表示
- 支払い予定 = 到着日の翌々月末（「7/31」のような表示）
