# 今日やったこと（かんたん説明）

---

## 一言でいうと
**「請求書が届いたら自動でスプレッドシートに入力してくれるロボット」を作りました。**

---

## 今まで（手作業）

```
メールを開く
  ↓
PDFを確認する
  ↓
金額・名前・日付などを手でスプレッドシートに入力する
  ↓
（毎回これを繰り返す…）
```

## これから（自動）

```
メールが届く
  ↓
ロボットが1時間ごとに自動チェック
  ↓
PDFをAI（Gemini）が読んで内容を理解
  ↓
スプレッドシートに自動入力される
```

---

## 便利なポイント

| こんなとき | ロボットが自動でやること |
|-----------|----------------------|
| 請求書メールが届いた | PDFを読んで名前・金額・日付を入力 |
| 5日より遅く届いた | 「⚠️締切超過」と書いて通知メールを送る |
| 新しい月になった | 新しいシートを自動で作る |
| 4月（新年度）になった | 新しいスプレッドシートを自動で作る |
| 読み取り失敗したとき | 「要確認」と書いてエラーメールを送る |
| 同じメールを二重処理しそうなとき | 処理済みタグをつけてスキップする |

---

## 今日の作業内訳

1. **設計** → どんな動きをするか仕様書を読み込んだ
2. **プログラム作成** → 5つのファイルにコードを書いた

4. **あとは設定するだけ** → メールアドレスを1箇所入力すればすぐ動く状態にした
